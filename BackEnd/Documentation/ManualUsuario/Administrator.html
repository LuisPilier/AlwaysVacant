¿Como usar el perfil de Administrator ?

Always Vacant

Este usuario con permisos super especiales se encarga de dar mantenimiento al contenido de la pagina y resolver problemas o correcciones por errores humanos .






Pantallas de Administrator

Pantalla Principal

<
Para registrarse presiona el boton de Sign In Y para iniciar sesion presiona el boton de Log In

Pantalla Login

En el campo   Username   debes introducir el nombre de usuario y en campo   Password  debes introducir la contraseña de tu cuenta y presiona   Login.

Pantalla Vacants


Aqui debes situar el cursor en la vacante que estas interesado y hacer click para abrir los detalles.

Pantalla Numbers of Jobs


Ingresa la cantidad de vacantes que deseas mostrar en la tabla de la pantalla de vacantes

Pantalla Select Jobs


Clickea en la vacante que deseas   remover o editarAssets

Pantalla Remove or Edit Jobs

Ingresa los datos que deseas editar y luego clickea el boton de   EDIT VACANT   para remover solo presiona el boton de  DELETE VACANT

Pantalla Select Category

Clickea en la categoria que deseas   remover o editar..

Pantalla Edit or Remove Category


Ingresa los datos que deseas editar y luego clickea el boton de   EDIT CATEGORY   para remover solo presiona el boton de  DELETE CATEGORY
