Propósito

Objetivo

¿Cuál es el objetivo de Always Vacant?

Always Vacant

El objetivo general del proyecto consiste en desarrollar un sitio web de una bolsa de empleos, el cual debe proveer en su plataforma empleos para los usuarios los cuales son ofrecidos por las empresas que necesitan trabajadores capacitados para sus vacantes. Del mismo modo facilitar el proceso de empleabilidad de una manera segura y rápida la cual se desarrolla no solo a nivel nacional sino internacional dándole oportunidad laboral a las personas de diferentes orígenes.
